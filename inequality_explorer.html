01 — Measure

What is the Gini coefficient?

The Gini coefficient is the most widely used measure of income inequality. It ranges from 0 (perfect equality) to 1 (one person has everything). Explore how it varies across the world and what it looks like graphically through the Lorenz curve.

How to read this

The diagonal line represents perfect equality. The further the Lorenz curve bows away from it, the more unequal the distribution. The Gini coefficient is twice the shaded area between the two lines.

Interpreting the scatter plot

Each dot is a country. The correlation coefficient (r) shown above the chart tells you the strength and direction of the linear relationship. Remember: correlation does not imply causation. Think about what other variables might explain patterns you observe — these are potential confounders for your regression analysis.

R code — replicate this analysis
# Load the exported CSV and run correlation + regression
data <- read.csv("inequality_correlates.csv")

# Scatter plot
plot(data$x_var, data$gini,
     xlab="X variable", ylab="Gini coefficient",
     main="Gini vs X", col="steelblue", pch=19)

# Correlation
cor.test(data$gini, data$x_var, method="pearson")

# Simple regression
model <- lm(gini ~ x_var, data=data)
summary(model)

Why do the measures differ?

The Gini is sensitive to changes in the middle of the distribution. The Palma ratio (top 10% share ÷ bottom 40% share) focuses on the extremes and is less affected by middle-income changes. Top 1% share captures the very wealthy — often missed in household surveys. When these measures diverge, it tells us something important about where in the distribution inequality is changing.

R code — compare measures
# After exporting the comparison CSV
data <- read.csv("inequality_measures.csv")

# Correlation between measures
cor(data[,c("gini","palma","top10","top1")], use="complete.obs")

# Plot all measures
library(ggplot2)
library(tidyr)
long <- pivot_longer(data, c(gini,palma,top10,top1))
ggplot(long, aes(year, value, color=name)) +
  geom_line() + facet_wrap(~name, scales="free_y")

R code — time series analysis
# Load exported trend data and plot with ggplot2
library(ggplot2)
data <- read.csv("inequality_trends.csv")

ggplot(data, aes(x=year, y=gini, color=country)) +
  geom_line(size=1.2) +
  geom_point(size=2) +
  geom_vline(xintercept=c(2008, 2020), linetype="dashed", alpha=0.5) +
  labs(title="Gini over time", x="Year", y="Gini coefficient") +
  theme_minimal()

# Test for trend (Mann-Kendall test)
library(trend)
uk <- data[data$country=="GBR",]
